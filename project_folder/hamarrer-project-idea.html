The Idea

For my individual project in IST263 Fall 2019, I would like to create a color and style guide webpage for a fictional wedding.
This webpage, first and foremost, will attempt to be W3C compliant.
The reason i've chosen this as my project is because I feel it would be the perfect opportunity to learn to create interactive pages using HTML, while also practicing design and branding strategies.
The ultimate goal of this webpage is to emulate a cross between a wedding invitation website and a graphic design style guide.

The Pages

The goal as of right now is to have a minimum of 6 pages

Page 1:
The Invite. The goal of this page is to welcome the wedding guests to the website and the event.
Page 2:
This page will detail the wedding colors and style elements with respect to decorations and the Dos and Donts
Such as:
  • Chosen representative logo
  • Wedding cards
  • table placards
  • bouquets
  • bridesmaids gifts
  • etc
Page 3:
This page will detail the wedding colors/style guides for attire (and its Dos and Donts).
Page 4:
This page will be where prospective attendees can send in their RSVPs
